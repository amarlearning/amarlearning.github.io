Handsontable Example Github : @amarlearning

Basic features are:

  • 1. Responsive table adjust itself with parent container.
  • 2. Each cell can be edited by just clicking on it.
  • 3. Data can be easiely copy/pasted via external CSV.
  • 4. Context Menu is avilable for options like add new row, coloum, copy, paste, alignment, remove row or coloum.
  • 5. Searching the entire table can be done easiely.
  • 6. Sizing handle can be used to change the size of column or row.
  • 7. Sort data in ascending or descending order throughout the column.
  • 7. Max row and coloum is set to 100 as required.
Note: Right click on the table for context menu.
